# POE2 国服集市监控网页测试版 v1.7

本版把多物品监控改为“项目包”逻辑：先在左侧填写一个物品的集市链接、通货、价格区间和动作设置，然后点击 **添加为监控项目**。右侧会生成一个独立项目包；勾选多个项目包后点击 **启动勾选项目包** 即可同时监控。

## 主要更新

- 多物品监控不再混在一个输入框里，而是按项目包管理。
- 每个项目包保存自己的链接、价格区间、通货筛选、检查间隔、提醒和前往藏身处动作。
- 支持每种通货设置不同价格范围，例如神圣石 1-30、崇高石 50-200。
- 如果在“批量快速添加”里一行填写一个链接，点击添加时会拆成多个独立项目包，便于勾选和删除。
- Cookie 仍只在当前页面使用，不会保存进项目包。

## 启动

```bash
npm start
```

打开：

```text
http://127.0.0.1:5177
```

如果端口被占用，先关闭旧版 npm start 窗口，或执行：

```bat
netstat -ano | findstr :5177
taskkill /PID 查到的PID /F
```

## 点击助手

保持国服集市页打开，并重新安装当前版本脚本：

```text
http://127.0.0.1:5177/poe2-cn-hideout-clicker.user.js
```

本工具不会自动购买，不会确认商人页签交易，只负责监控、提醒和发送“前往藏身处”点击任务。
